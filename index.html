FlexBox

Généralité

La Flexbox est un mode de mise en page permettant de disposer les éléments de manière flexible et de gérer leur alignement quelque soit la taille de l'écran. Chaque propriété s'écrit dans la zone CSS concernée. Sur la compatibilité, les flexbox fonctionnent avec tous les navigateurs hormis: - IE9 - IE10, il faut utiliser la syntaxe de 2011 - IE11, comporte quelques bugs https://developer.mozilla.org/fr/docs/Web/CSS/Disposition_des_bo%C3%AEtes_flexibles_CSS/Utilisation_des_flexbox_en_CSS
