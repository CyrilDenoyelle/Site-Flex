DISPLAY

Propriété indispensable sur l'élément englobant les enfants à piloter. Elle active la propriété flex sur tous les enfants du conteneur. syntaxe: .parent{ display:flex ou inline-flex; } Cela placera tous les enfants côte à côte en essayant de remplir tout l'espace. Flex définit le conteneur comme élément de type block. Inline-flex définit le conteneur comme élément de type inline. http://www.alsacreations.com/xmedia/tuto/exemples/flexbox/flextest01.html

ORDER

Propriété d'ordonnancement permettant de classer les éléments suivant un ordre défini sans modifier le fichier html. syntaxe: .enfant{ order: nombre; } Sa valeur initiale est égale à 0. Les valeurs les plus fortes se trouveront en fin de pile. On peut utiliser un nombre positif ou négatif. http://www.alsacreations.com/xmedia/tuto/exemples/flexbox/flextest07.html

ALIGN-SELF

A les mêmes propriétés que ALIGN-ITEMS mais elle permet de gérer l'alignement d'un seul élément de manière indépendante. syntaxe: .enfant{ align-self: auto ou flex-start ou flex-end ou center ou baseline ou stretch; http://www.alsacreations.com/xmedia/tuto/exemples/flexbox/flextest05b.html
